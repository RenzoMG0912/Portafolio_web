# Portafolio - Estructura Organizada

Este proyecto fue reorganizado para separar responsabilidades y mantener todo unido bajo una estructura clara para React.

## Estructura

- src/components/Portfolio.jsx: componente principal del portafolio
- src/data/portfolioData.js: contenido y datos (links, skills, proyectos, textos)
- src/hooks/useInView.js: hook reutilizable para animaciones por visibilidad
- src/styles/portfolio.css: estilos globales del portafolio
- src/index.js: punto de exportacion del componente
- preview/portfolio_preview.html: version HTML de referencia visual

## Nota

Si usas Vite o CRA, importa el componente desde `src/components/Portfolio.jsx` o desde `src/index.js`.
